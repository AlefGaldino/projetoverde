O projeto Delivery de Combustível tem como objetivo simular um sistema de entrega de combustível, onde o usuário pode solicitar gasolina, etanol ou diesel e receber o produto no local desejado.

O sistema no aplicativo também simula controle de estoque, pedidos e logística de entrega.
-

⚙️ Como o sistema funciona

1. O usuário faz um pedido informando:

Tipo de combustível

Quantidade desejada

2. O sistema verifica o estoque disponível

3. Se houver estoque suficiente:

O pedido é aprovado

A entrega é iniciada

4. O combustível é “enviado” até o destino (simulação)

5. O status do pedido é atualizado:

Pendente

Em entrega

Entregue

📊 Fluxo do sistema
Usuário → Pedido → Validação → Entrega → Finalização
